# `.DollarNames.DataFrame`

## Usage

```r
list(list(".DollarNames"), list("DataFrame"))(x, pattern = "")
```


## Arguments

Argument      |Description
------------- |----------------
`x`     |     DataFrame
`pattern`     |     code-stump as string to auto-complete
