# Lazy with column

## Format

An object of class `character` of length 1.

```r
LazyFrame_with_column(expr)
```

## Arguments

- `expr`: one Expr or string naming a column

## Returns

A new `LazyFrame` object with add/modified column.

add or replace columns of LazyFrame
