# `print_env`

print recursively an environment, used in some documentation


## Description

print recursively an environment, used in some documentation


## Usage

```r
print_env(api, name, max_depth = 10)
```


## Arguments

Argument      |Description
------------- |----------------
`api`     |     env
`name`     |     name of env
`max_depth`     |     numeric/int max levels to recursive iterate through
