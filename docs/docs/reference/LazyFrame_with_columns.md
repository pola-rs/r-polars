# `with_columns`

Lazy with columns

## Description

add or replace columns of LazyFrame

## Usage

```r
LazyFrame_with_columns(...)
```

## Arguments

| Argument | Description                               | 
| -------- | ----------------------------------------- |
| `...`         | any single Expr or string naming a column | 

## Value

A new `LazyFrame` object with added/modified columns.
