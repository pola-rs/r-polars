# `alias`

Alias

## Description

Change name of Series

## Format

An object of class `character` of length 1.

## Usage

```r
Series_alias(name)
```

## Arguments

| Argument | Description              | 
| -------- | ------------------------ |
| `name`         | a String as the new name | 

## Value

Series

## Examples

```r
pl$Series(1:3,name = "alice")$alias("bob")
```
