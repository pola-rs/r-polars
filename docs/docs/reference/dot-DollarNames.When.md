# `.DollarNames.When`

auto complete $-access into a polars object


## Description

called by the interactive R session internally


## Usage

```r
list(list(".DollarNames"), list("When"))(x, pattern = "")
```


## Arguments

Argument      |Description
------------- |----------------
`x`     |     When
`pattern`     |     code-stump as string to auto-complete
