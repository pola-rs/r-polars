# `tail`

Tail a DataFrame

## Description

Get the last n rows.

## Usage

```r
DataFrame_tail(n)
```

## Arguments

| Argument | Description                                             | 
| -------- | ------------------------------------------------------- |
| `n`         | positive numeric of integer number not larger than 2^32 | 

## Details

any number will converted to u32. Negative raises error

## Value

DataFrame
