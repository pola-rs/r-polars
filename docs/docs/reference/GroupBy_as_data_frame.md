# `GroupBy_as_data_frame`

convert to data.frame


## Description

convert to data.frame


## Usage

```r
GroupBy_as_data_frame(...)
```


## Arguments

Argument      |Description
------------- |----------------
`...`     |     any opt param passed to R as.data.frame


## Value

R data.frame


## Examples

```r
pl$DataFrame(iris)$as_data_frame() #R-polars back and forth
```
