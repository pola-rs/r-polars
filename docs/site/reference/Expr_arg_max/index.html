    data

    Index of min value

    Format

    a method

    Expr_arg_max

    Returns

    Expr

    Get the index of the minimal value.

    Details

    See Inf,NaN,NULL,Null/NA translations here docs_translations

    Examples

    pl$DataFrame(list(
  a = c(6, 1, 0, NA, Inf, NaN)
))$select(pl$col("a")$arg_max())
#> polars DataFrame: shape: (1, 1)
#> 
#>  a   
#>  --- 
#>  u32 
#> 
#>  4   
#>
